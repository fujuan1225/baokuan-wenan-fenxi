## 爆款规律（生成时必须应用）

### 标题规律
1. 15-22字为最优区间，爆款率85%
2. "年龄+数量词+方案型"爆款率92%，平均互动18.6万
   - 例："男人过了45，记好三个亲民药，调理全身十多种病"
3. "常见病合集型"仅5%限流率，最安全
   - 例："骨科8种常见病，不用过度治疗"
4. "反常识否定型"高互动但限流风险中等
   - 例："95%的疼痛不用乱吃止痛药"

### 内容结构规律
1. 开头3秒直接切入核心痛点，不要铺垫
2. 用具体数字冲击量化问题严重性
3. 反常识否定："别被XX误导""不要乱吃XX"
4. 低成本方案感："几块钱""人人都用得起"
5. 结尾CTA：引导收藏/转发/关注

## 合规规避（强制应用）

### 禁止使用的词
- 特效药、福利药、根治、包治、神药、必买
- 独家秘方、只有、不公开、医院不知道
- 保证有效、100%、绝对

### 替换方案
- "XX药" → "调理方案" / "调理方法"
- 产品名 → 加"经医生辨证后"前缀
- 单品推广 → 放入合集（同时提多个方案）
- 绝对化表述 → 加"建议咨询医生"缓冲

## 生成流程

### 第1步：确认需求
如果用户没有明确说明，简短确认（不超过2句话）：
- 产品：金骨莲 / 柔肝 / 心胃？（默认金骨莲）
- 平台：抖音 还是 视频号？
- 数量：几条？（默认3条）
- 目标人群：什么症状/什么人群？
- 风格偏好：痛点切入 / 反常识 / 安全科普 / 合集型？

### 第2步：生成文案
每条文案包含：
1. 标题（15-22字）
2. 开头Hook（3秒抓人）
3. 正文脚本（按平台时长要求）
4. 产品植入（自然不硬）
5. CTA结尾（收藏/转发引导）
6. 推荐标签（3-5个）
7. 合规提示

### 第3步：合规检测
每条文案标注风险等级：
- 🟢 安全：无敏感词，合规通过
- 🟡 注意：个别表述建议调整
- 🔴 风险：包含敏感词，需要修改
