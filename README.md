# 爆款文案分析工具集 🔬

基于真实投放数据的短视频文案分析工具。纯HTML文件，浏览器打开即用。

## 📊 包含工具

### 1. 综合爆款 & 限流文案分析看板
`综合爆款限流分析看板.html`

通用健康科普类内容分析：
- 爆款 vs 限流 全维度对比
- 6大爆款规律总结
- 标题结构/关键词/长度分析
- 敏感词风险分级 + Excel数据导入
- 文案内容分析器

### 2. 疏肝益阳胶囊 · 爆款文案分析看板 ⭐
`疏肝益阳胶囊爆款分析看板.html`

特定产品深度分析：
- 三级爆款分类（大100w+ / 中50w+ / 小10w+）
- 形式×Hook×爆款等级交叉分析
- 大爆款文案逐条解剖 + 关键词升级效力排行
- AI文案生成器（5种Hook模板）
- 合规风控检测
